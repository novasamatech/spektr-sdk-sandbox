# Spektr SDK sandbox (test environment)

Dev environment for running and testing your Polkadot Dapps.

## Features:

- Integration with extension wallets — test your injectWeb3 injections with real wallets.

## Roadmap:

- PAPI integration check. Redirect your requests from dapps to host.
- Different dapp representation — widgets, chat widgets, etc.
- Dapp navigation history

## Local development

```shell
npm install
```

```shell
npm run dev # local instance
```
